# Prompt Templates

Basis-Templates für die Erstellung neuer Prompts. Kopiere ein Template als Vorlage für dein Projekt.

## Verfügbare Templates

- **browser-navigation.md** - Für URL-Navigation mit Auth
- **api-analysis.md** - Für API-Call Analyse  
- **network-monitoring.md** - Für Network-Traffic Überwachung
- **data-extraction.md** - Für strukturierte Datenextraktion

## Wie man ein Template verwendet

```
1. Template aus diesem Ordner öffnen
2. Kopie erstellen: cp [template].md ../active/[projekt]_PROMPT.md
3. Für dein Projekt anpassen
4. In ../active/ speichern
```
